Nike Sales Analysis Dashboard using Power BI

Overview
This project presents a comprehensive sales analysis of Nike products using Power BI. The dashboard provides insights into revenue, profit, units sold, product performance, and regional sales trends through interactive visualizations.

Objectives
- Analyze Nike sales performance.
- Identify high-performing products.
- Compare sales across regions.
- Track revenue and profit trends.
- Provide business insights through data visualization.

Tools Used
- Power BI
- Microsoft Excel / CSV
- Data Analysis
- Data Visualization

Dataset Information
The dataset contains:
- Order ID
- Product Name
- Product Line
- Gender Category
- MRP
- Revenue
- Profit
- Units Sold
- Region
- Date Information

Dashboard Features

KPI Cards
- Total Revenue
- Total Profit
- Total Units Sold
- Product Count

Visualizations
- Profit by Region
- Revenue by Product Line
- Units Sold by Gender
- Product-wise Profit Analysis
- Year-wise Product Sales Trend

Filters
- Gender Category
- Product Line
- Date Filter

Key Insights
- Lifestyle products generated the highest revenue.
- Profit varies significantly across regions.
- Product sales trends show seasonal variations.
- Men's and Women's product categories contribute major sales volume.
